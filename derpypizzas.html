Role

I was the sole developer on this project.


Technologies Used

  • Laravel - An MVC Framework written in PHP. This was used to communicate with the database, configure web routes and authentication.
  • Heroku - For hosting both the front-end and database(PostgreSQL).
  • Bootstrap - A CSS Framework to help with styling the website.

    • Lessons Learned

    This was the second project I built with Laravel and as such, served as a refresher. This was my first time deploying a Laravel application on my own, without an instructor's help. The project started off with an online tutorial, but then I added in extra functionality such as reviews of the restaurant. To be added in the future, is an admin panel where an admin can add menu items.
