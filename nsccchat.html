Role

In this project, I was on a team of 5 developers. My biggest contribution to the project was the collapsible navigation bar to the left which would fetch all of the messages of the respective chatroom when clicked on.


Technologies Used

  • React - A Javascript Framework based off of components.
  • Firebase - A NoSQL Database based in the cloud, used to house all data for the project.
  • Bootstrap - A CSS Framework to help with styling the website.
  • Material UI - Another CSS Framework which works very well with React to build the navigation.

    • Lessons Learned

    This was the second time I had worked with a team of developers. As such, this helped reinforce the lessons of using Git and writing readable and maintainable code for my team members. Additionally, this project helped reinforce the basics of React. Particularly, the piece of unidirectional flow of data.
